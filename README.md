Customer Behaviour Analysis is a data analytics project that performs Exploratory Data Analysis (EDA) on 3,900+ customer records using Pandas and NumPy to uncover purchasing patterns, customer segments, and behavioural trends. Missing values were handled using median imputation due to outliers. An interactive Power BI dashboard was built to visualize customer retention, spending behaviour, and demographic insights. SQL queries were optimized to reduce data retrieval and reporting time by approximately 30%.
